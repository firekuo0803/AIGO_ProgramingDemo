# AIGO_ProgramingDemo  
  
這是AIGO作品的DEMO程式  
  
2.py影片拆幀程式  
cut.py是循環自動切割圖片程式  
label.py是安裝labelImg標註圖片程式的程式  
MotionCap.py是追蹤動作並儲存圖片的程式  
MotionCap2.py是追蹤動作並顯示的測試程式  
measure.py是臉部測距程式  
radiotest.py是圖片切割程式  
poseAI.ipynb是辨識姿勢的AI程式(最後兩個cell不要理他，那是失敗的在googlecolab跑影片)  
  
    
tell_images是別的專案的東西借來用別理他
